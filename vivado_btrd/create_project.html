Xylon ZedBoard ZYNQ Base TRD - recreating project from scratch

1. TCL script:
You can use a TCL script to recreate and run the project. When run from the command line, the script adds sources, sets properties, recreates HDL libraries and runs processes. You can use this script to do the following:
  • Create a new project and run implementation or other processes from the command line
  • Set properties in other projects to desired values
  • Modify the script to change design properties, and then run processes

2. How to create project from scratch?
  • To create/recreate project you must have evaluation license for logiCLK logicBRICKS IP core. Please contact Xylon sales for further details (mailto:support@logicbricks.com).
  • 1. Run Vivado Design Suite Tcl Shell
    • Use the following command to invoke the Vivado Design suite Tcl Shell either at the Linux command prompt or within Windows cmd window:
      vivado -mode tcl
    • On Windows, select Start > All Programs > Xilinx Design Tools > Vivado 2014.4 > Vivado 2014.4 Tcl Shell)
    • Change directory into vivado_btrd/scripts (Note: Tcl Shell uses slash "/" for positioning)
    • Run following command : source project.tcl and wait for Vivado to generate project.
  • 2. Open generated Vivado project file in vivado_btrd/project directory (vivado_zedboard_btrd.xpr file) and regenerate output products for block design, etc.
